Subversion (svn) download and build instructions for j3d-core

To build the j3d packages, you must checkout the following three svn repositories:

  • j3d-core
  • j3d-core-utils
  • vecmath

These three top-level directories must be named exactly as shown above and they must be sibling directories. To ensure this, run the svn checkout command for each of the respositories from the same parent directory. For example:

    cd <j3d-root-dir>
    svn checkout https://vecmath.dev.java.net/svn/vecmath/trunk vecmath
    svn checkout https://j3d-core.dev.java.net/svn/j3d-core/trunk j3d-core
    svn checkout https://j3d-core-utils.dev.java.net/svn/j3d-core-utils/trunk j3d-core-utils

After you have downloaded the three svn repositories, read the README-FIRST.txt file, then read and follow the instructions in the README-build.html files in all three svn repositories.

Go to the "Subversion" page for instructions on how to access the j3d-core source code. Automated svn change messages are sent to the commits 'at' j3d-core.dev.java.net list list. Click here to subscribe to this list.
